Supabase Utility

Use this tool to test your connection and get the SQL commands needed for the Familia app.

1. Required SQL Schema

Go to your Supabase project -> SQL Editor -> New Query, and run this exact code to create the table your app needs:

CREATE TABLE familia_data (
  key TEXT PRIMARY KEY,
  value TEXT NOT NULL,
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Optional: Set up Row Level Security (RLS) to allow anon access for testing
ALTER TABLE familia_data ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow anonymous read/write" ON familia_data FOR ALL USING (true) WITH CHECK (true);
